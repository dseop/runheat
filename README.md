# 문서 인덱스

이 문서는 저장소 문서의 진입점입니다. 기능 상세를 중복해서 적지 않고, 필요한 기준 문서로 이동하기 위한 인덱스로 유지합니다.

## 제품/개발

- `requirements.md`: 제품/화면 요구사항의 단일 기준 문서
- `planning.md`: 제품 방향과 사용자 흐름 요약
- `development.md`: 코드 구조, 핵심 타입, 유지보수 포인트
- `healthkit_data.md`: HealthKit 러닝 데이터 필드와 조회 파이프라인 참고

## 기능 계획

- `plan_fetch_splits.md`: 1km 스플릿 기능 검토용 참고 계획

## 출시/운영

- `app_store.md`: App Store Connect 문구, 심사 메모, 릴리즈 노트 작성 템플릿
- `app_store/`: 버전별 App Store 스크린샷 또는 제출 참고 자료 보관
